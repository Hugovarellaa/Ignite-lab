## ⚙️ Instalação

```
# Abra um terminal e copie este repositório com o comando
$ git clone https://github.com/Hugovarellaa/ignite-timer
```

```
# Acesse a pasta da aplicação
cd ignite-timer

# Instale as dependências
$ yarn

# Inicie a aplicação back-end
$ yarn dev
```
